git-order

发表于 2018-04-17 | 分类于 git

git常用命令

  • git init 初始化本地仓库
  • git config --global user.name 配置用户名
  • git config --global user.email 配置邮箱
  • git config user.email 查看邮箱
  • git add . 添加到 暂存区
  • git commit -m 备注信息 提交到本地仓库
  • git status 查看仓库状态
  • git remote add origin 地址 把远程仓库的地址记录到一个变量上 origin
  • git remote -v 查看远程仓库
  • git push -u origin master 第一次推送到远程仓库 origin
  • git push 假如第一次使用 -u 参数 后期 直接使用该命令进行推送
  • git clone 地址 从地址克隆 远程仓库到本地电脑上
  • git pull 获取更新
  • git checkout 文件名 还原文件到上一个commit的状态
  • git reset --hard 版本号 还原到任意版本
  • git reset --hard HEAD^ 还原到上一个版本
  • git log 查看日志信息
  • git log --oneline 精简查看日志
  • git reflog 查看仓库版本操作历史
  • git branch 分支名 创建分支
  • git branch 查看分支
  • git merge 分支名 合并分支
  • git branch 分支 -d 删除分支
  • git checkout 分支名 切换分支
  • git 文件忽略清单 .gitignore 在文件里面添加要忽略的文件名 支持正则语法

h5c3-2D转换

发表于 2018-03-25 | 分类于 h5c3

2D转换（变换）transform

  • 常用函数

    translate(x,y) translateX(x) translateY(y)

  • 使用方式：为元素添加transform属性，如transform:translate(100px,100px)

  • 细节
    • 类似于定位，它并不会影响其它的元素
    • 它是参照元素自身的位置
    • 对行内标签没有效果
  • 应用：实现定位元素的水平垂直居中
    • 定位的百分比是参照父容器的宽高
    • translate的百分比是参照元素自身的宽高
