h5c3-2D转换

发表于 2018-03-25 | 分类于 h5c3

2D转换（变换）transform

  • 常用函数

    translate(x,y) translateX(x) translateY(y)

  • 使用方式：为元素添加transform属性，如transform:translate(100px,100px)

  • 细节
    • 类似于定位，它并不会影响其它的元素
    • 它是参照元素自身的位置
    • 对行内标签没有效果
  • 应用：实现定位元素的水平垂直居中
    • 定位的百分比是参照父容器的宽高
    • translate的百分比是参照元素自身的宽高

2d变换 - rotate

作用：可以让元素在2维坐标系中旋转

  • 特点：围绕z轴进行旋转
  • 旋转的方向：正值 》》顺时针方向旋转 负值》》逆时针方向旋转

  • 使用方式

    为元素添加变换属性transform

    设置变换属性的值为rotate,如:transform:rotate(90deg)

  • 变换圆心点：transform-origin：
  • 关键字：top left right bottom center
  • 坐标：px
  • 该属性可以修改元素旋转的时候的中心点
    1. transform-origin:50% 50%; 默认值 元素的中心位置 百分比是相对于自身的宽度和高度
    2. transform-origin:top left; 左上角 和 transform-origin：0 0;相同
    3. transform-origin:50px 50px; 距离左上角 50px 50px 的位置
    4. transform-origin：0; 只写一个值的时候 第二个值默认为 50%;

2d变换 - scale

作用：实现元素在2维坐标系中的绽放

  • 使用方式

    为元素添加变换属性：transform

    为元素的变换属性设置值：scale,如：transform:scale(2,3)

  • 清除添加的变换效果：
    • transform: none
  • 添加图片之后的问题
    • 会有默认的基线距离–6px
    • 问题：如何清除图片的基线距离
      • vertical-align:bottom
      • 将font-size设置为0，它需要添加给父容器
      • 将图片设置为块级元素：display:block
