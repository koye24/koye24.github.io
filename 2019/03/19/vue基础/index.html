关于vue介绍

1.vue并不是前端框架里面唯一的解决方案，（react,angular）, 这几大框架原理都是一样的，都是用来开发单页应用

2.单页应用只有一个html文件，里面的模块都是一个div，vue里面模块也叫做组件，组件的切换需要依赖路由

MVVM设计模式

1.m 代表数据 model

2.v 代表模板界面 view

3.vm 代表把数据渲染到模板 viewmodel

注意：mvvm只是一个设计模式，只是一个概念，需要用到时候再去背比如面试时候，同样遵循mvvm设计模式框架还有很多，比如angular

指令

  • v-text - 往标签里面插入字符串文本
  • v-html - 往标签里面插入html富文本字符串
  • v-bind - 给标签属性绑定一个data动态值，可以缩写成：
  • v-for - 列表循环，列表循环有三种结构
  • v-model - 用于双向数据绑定，可以在模板中修改data的数据，通常用于表单元素
  • v-if/v-show - 用于条件渲染，v-if在隐藏元素时候不会把div生成到dom，v-show使用css样式隐藏
  • v-on - 用于绑定事件，可以缩写成@符号
