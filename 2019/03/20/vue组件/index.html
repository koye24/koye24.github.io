vue组件

发表于 2019-03-20 | 分类于 vue

创建组件的两种方式

1.局部组件

2.全局组件

局部组件

1.声明一个对象。对象可以是template属性声明组件的模板

2.需要在根实例中调用局部组件

3.在页面模板中使用组件名渲染组件

全局组件

1.直接使用Vue.component创建全局组件，第一参数是组件名，第二个参数是配置对象,template配置模板

2.在页面模板中使用组件名渲染组件

组件的属性

组件中的data声明必须是一个函数，在函数里面必须返回一个对象，对象内部可以声明组件的动态数据

注意：

在组件中除了data的使用方式和根实例一样之外，其他属性配置和根实例组件都是一样，包括

组件传值

在之前组件的数据只能通过两种方式声明，data/computed，目前组件中可以可以通过props接收外部传递过来的数据

1.在组件中声明可以接收数据

2.组件可以多次调用，并且通过属性传递进来不同的值

组件传递事件

组件除了可以接收传递过来的属性之外，还可以接收传递的事件

1.在根实例中声明一个事件,把该事件传递到组件

2.在根实例模板中把事件传递给组件,addcart是传递的事件名

3.在组件中手动触发传递过来的事件

子组件传值给父组件

1.所谓的子组件传值给父组件，无非就是在子组件中触发传递过来的事件时候，通过this.$emit的第二个参数传递参数

2.根实例中的事件处理函数可以接收子组件的数据

兄弟组件传值

兄弟组件指的是两个同级组件之前的数据传递，

1.找组件公共的介质，这是介质就是根实例组件，根实例组件可以包含公共的事件或者数据

2.先把第一个组件的数据传给父组件

3.父组件再把值传给另外一个组件
